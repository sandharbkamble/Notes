IRCTC functional requirement
- Able to login and signup
- fetch / search trains from A to B
- Show available seats
- Book ticket
- Download ticket


LLD:

- hashed password
- Tickets List (booked)
- userID


Ticket entity

- String userID
- String source
- String destination

Train entity
- trainID
- trainNo
- DateTime departtime
- DateTime arrivetime
- seats booked in a matrix in 1 and 0 (2d list)
